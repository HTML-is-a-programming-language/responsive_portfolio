약 6개월 간 국비지원 교육을 받은 후 만든 포트폴리오입니다.

웹 표준과 웹 접근성을 지키며 반응형 웹 사이트로 작업하려고 노력했으며 HTML, CSS, JQuery를 사용하여 작업했습니다.

주로 구현한 기능
- hover() : 시각적인 효과
- click() : 네비게이션 클릭 시 스크롤 효과, 메뉴 또는 이미지 애니메이션 효과, 탑버튼 효과
- click(), setInterval(), clearInterval() : 슬라이드 효과
- scroll() : 스크롤 위치에 요소들의 스타일 동적으로 변경
- animate() : 애니메이션 효과
